# BG Document Generator - Setup Instructions

## Quick Start

### 1. Install Python Dependencies
```bash
pip install python-docx
```

### 2. Prepare Your Input File
- Copy all your BGs (generated by ChatGPT) into a text file named `bgs_input.txt`
- The script supports multiple formats (see below)

### 3. Run the Generator
```bash
python bg_generator.py
```

### 4. Collect Your Documents
All Word documents will be generated in the `BG_Documents` folder with names like:
- `BG_001.docx`
- `BG_002.docx`
- `BG_003.docx`
- ... and so on up to BG_150.docx

---

## Supported BG Formats

The script automatically detects and handles these formats:

### Format 1: Numbered BGs (RECOMMENDED)
```
1. First background content here...
2. Second background content here...
3. Third background content here...
```

### Format 2: Dash/Line Separated
```
First background content here...
---
Second background content here...
---
Third background content here...
```

Or use `===` or `___` as separators.

### Format 3: Blank Line Separated
```
First background content here...

Second background content here...

Third background content here...
```

---

## Customization

Edit `bg_generator.py` to modify:
- **Document formatting** (font, size, colors, spacing)
- **Title and headers** (change "Background" to something else)
- **Output folder name** (change `BG_Documents` to your preferred name)
- **Document naming** (change `BG_{idx:03d}` to your preferred naming scheme)

---

## Tips for Best Results

1. **Use numbered format** - It's the most reliable
2. **Keep BGs together** - Don't split a single BG across multiple lines artificially
3. **Paste directly from ChatGPT** - The script handles formatting automatically
4. **Batch processing** - For 150 BGs, it typically takes 5-10 seconds
5. **Version control** - Keep your `bgs_input.txt` as a backup

## Example Customizations

### Add Date/Timestamp to Documents
Edit the `create_word_document()` function:
```python
from datetime import datetime
doc.add_paragraph(f"Created: {datetime.now().strftime('%Y-%m-%d')}")
```

### Add Page Numbers
```python
section = doc.sections[0]
footer = section.footer.paragraphs[0]
footer.text = "Page "
```

### Change Styling
```python
# Modify fonts, sizes, colors, etc.
for run in paragraph.runs:
    run.font.size = Pt(14)  # Change to your preferred size
    run.font.bold = True
    run.font.color.rgb = RGBColor(0, 0, 139)  # Dark blue
```

## Troubleshooting

**Q: Script says "No BGs found"**
- Check your separator format matches one of the supported formats
- Ensure there's no trailing whitespace or extra formatting

**Q: Some BGs are empty**
- Check for extra blank lines or inconsistent separators
- Try the numbered format for better reliability

**Q: Want to modify formatting later?**
- The generated `.docx` files are standard Word documents
- You can open and edit them normally in Microsoft Word
- Or modify the script and regenerate
